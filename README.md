
# Laboratorio N°1 – Anemometría de Hilo Caliente
#### Facultad de Ingeniería, Universidad Nacional de la Plata
##### Mecánica de Fluidos II

---


    
### Introducción:
En este laboratorio vamos a visualizar, validar y determinar mediante métodos numéricos distintos parámetros asociados al flujo turbulento. Esto se hará usando los datos de velocidad horizontal u y vertical v obtenidos en un ensayo de tunel de viento mediante un anemómetro de hilo caliente colocado a distintas alturas detrás de un tubo.
